## 📖 용어 사전 (Glossary)
분산에너지의 핵심 용어 해설 카드입니다. 본문 분석 시 해당 용어로의 링크 연결을 권장합니다.

* [VPP (가상발전소)](glossary/VPP.md) - 분산된 발전원을 소프트웨어적으로 통합 제어하여 하나의 발전소처럼 운영하는 기술.
* [ESS (에너지저장장치)](glossary/ESS.md) - 생산된 전력을 배터리 등에 저장했다가 필요한 시점에 공급하는 설비.
* [플러스DR](glossary/플러스DR.md) - 잉여 전력이 발생할 때 전력 소비를 늘리면 인센티브를 제공하는 수요 관리 제도.
