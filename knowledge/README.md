# 🗺️ 분산에너지 지식 지도 (Index)

이 문서는 지식 베이스 내에 축적된 핵심 지식을 효율적으로 탐색하고 LLM의 토큰 소모를 방지하기 위해 마련된 마스터 **Index 노트**입니다.

> [!IMPORTANT]
> **에이전트 판단 가이드 (토큰 효율화)**
> 1. 기사 분석 후 명제를 추가하려 할 때, 먼저 아래 **[1. 관점별 명제 현황판]**의 명제 목록을 검토하십시오.
> 2. 추가하려는 명제와 유사한 명제가 이미 리스팅되어 있다면, 해당 관점의 명제 노트(`knowledge/propositions/*.md`)를 절대 수정하지 않고 스킵하십시오.
> 3. 기존에 존재하지 않는 전혀 새로운 명제일 경우에만 해당 명제 파일을 열어 수정하십시오.
